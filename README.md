## üöÄ Getting Started

Follow these steps to clone and run the project on your local machine.

### Prerequisites

You need to have **Node.js** and **npm** or **Yarn** installed.

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Roya9022/donation-app.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd donation-app
    ```
3.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```
4.  **Run the development server:**
    ```bash
    npm run dev
    # or
    yarn dev
    ```
5.  Open your browser to: **[http://localhost:3000](https://www.google.com/search?q=http://localhost:3000)**

## üèóÔ∏è Current Status & Future Features

This project is a functional prototype. The following pages and features are currently implemented, with a large backlog of planned enhancements:

### Implemented Features

  * **Responsive Charity Listing:** View a feed of all charities.
  * **Category Filtering:** Filter charities using interactive chips (including the "All" chip).
  * **Full-Text Search:** Filter the list by keyword.
  * **Charity Detail Pages:** Dedicated pages for each charity with description and external links.
